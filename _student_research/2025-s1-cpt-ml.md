abstract_background: "Cone Penetration Testing (CPT) is one of the most widely used in-situ tools for soil profiling, yet conventional interpretation methods rely on empirical charts that may not capture the complexity of local soil conditions."
abstract_objective: "To develop a machine learning model that accurately classifies soil types from CPT data, improving upon traditional Robertson (1990) classification charts."
abstract_methodology: "A supervised classification framework is adopted using random forest and gradient boosting algorithms trained on a curated CPT database with laboratory-confirmed soil classes. Feature engineering includes normalized CPT parameters (Q_t, F_r, B_q) and depth-based context windows. Model performance is evaluated through stratified k-fold cross-validation."
abstract_contribution: "The proposed model is expected to provide higher classification accuracy than chart-based methods, particularly for intermediate and mixed soils, and will be delivered as an open-source Python tool for practitioners."
